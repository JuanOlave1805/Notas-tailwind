Colores

BG color Sintaxis: (bg-lime-100)
(bg) es el indicativo de que es background, es decir: color de fondo
(lime) es el indicativo del color
(100) es el indicativo de la tonalidad del color
Para dar el color a los textos se maneja igual solo que se escribe de esta manera: (text-sky-900) Para usar un color especifico se maneja de esta manera: text-[color en hexadecimal] , bg-[color en hexadecimal]
Para usar colores especificos como black, red, yellow ... se hace de esta manera: (text-white) (bg-red)

NOTA: el color debe ir de 100 en 100, al colocar un valor distinto no lo valida como color. Pero puedo usar 50 de valor el cual sera el color mas claro
NOTA: Para manejar colore se hace de la siguiente manera: "text-white" es decir text-color el color de la misma manera que con el background

Tambien se puede usar variables colocandolas en el archivo config.js de tailwind

Padding

El padding se usa de la siguiente manera:
(p-2) cuando queremos darle el padding en todos los lados, arriba, abajo, izquierda y derecha
(px-2) cuando queremos darle el padding en los lados, izquierda y derecha
(py-2) cuando queremos darle el padding en los lados, arriba y abajo
(ps-2) cuando queremos darle el padding en el lado izquierdo
(pe-2) cuando queremos darle el padding en el lado derecho
(pt-2) cuando queremos darle el padding en el lado arriba
(pr-2) cuando queremos darle el padding en el lado abajo
(pt-[10px]) cuando queremos usarlo personalizado o con una variable se hace de esa manera

NOTA: Para manejar el hover en los colores se hace de la siguiente manera "hover:bg-lime-800"; de esa manera se trabaja el hover

NOTA: Para manejar los gradientes en tailwind, es de la siguiente manera "bg-gradient-to-r from-color_inicial to-color_final ..."

NOTA: Para hacer un texto con con gradiente es de la siguiente manera:
"bg-clip-text text-transparent bg-gradient-to-r from-pink-200 to-violet-200"
